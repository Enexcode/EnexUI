How to Use EnexUI

Start using EnexUI in just a few steps. No complex setup — clean and flexible utility-first styling.

1. Add Class to HTML

Choose a component and apply the required class names to your HTML elements.

<button class="enx-btn-primary">Click me</button>
2. Include CSS

Download the CSS file or copy from CDN and link it in your HTML.

<link rel="stylesheet" href="https://cdn.enexcode.com/enexui.css">
3. Add JS (If Needed)

Some interactive components require JS. Copy script from docs or use our minified file.

<script src="https://cdn.enexcode.com/enexui.min.js"></script>

Every component has its own usage guide in the documentation.
