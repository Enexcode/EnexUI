Usage Instructions Source Code

You can use this scrollbar utility in two ways:

  1. Include the CSS file:
    2. <link rel="stylesheet" href="scrollbar_enexui.css" />
  2. Or copy the CSS code manually:
    3. .enexui_custom_scrollbar {
  --scrollbar-bg: #f1f1f1;
  --scrollbar-thumb: #f39200;
  --scrollbar-thumb-hover: #d57d00;
  --scrollbar-width: 8px;
  --scrollbar-radius: 10px;

  scrollbar-width: thin;
  scrollbar-color: var(--scrollbar-thumb) var(--scrollbar-bg);
}

.enexui_custom_scrollbar::-webkit-scrollbar {
  width: var(--scrollbar-width);
  height: var(--scrollbar-width);
}

.enexui_custom_scrollbar::-webkit-scrollbar-track {
  background: var(--scrollbar-bg);
}

.enexui_custom_scrollbar::-webkit-scrollbar-thumb {
  background-color: var(--scrollbar-thumb);
  border-radius: var(--scrollbar-radius);
}

.enexui_custom_scrollbar::-webkit-scrollbar-thumb:hover {
  background-color: var(--scrollbar-thumb-hover);
}
  3. Apply the class to any scrollable element:
    4. <div class="enexui_custom_scrollbar">...</div>

Customization Guide

Developers can customize this component easily by modifying CSS variables:

  • --scrollbar-bg: Background color of scrollbar track
  • --scrollbar-thumb: Color of scrollbar thumb
  • --scrollbar-thumb-hover: Hover color of thumb
  • --scrollbar-width: Width/height of scrollbar
  • --scrollbar-radius: Border radius of thumb
Advantages:
  • Modern and professional dark theme design
  • Fully customizable using CSS variables
  • Copy or download options for easy integration
  • Works on all modern browsers
  • Part of scalable EnexUI utility framework
