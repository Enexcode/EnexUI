Installation

To install EnexUI, simply include the CSS file in your HTML:

<link rel="stylesheet" href="enexui.min.css">

Or via CDN:

<link href="https://cdn.example.com/enexui.min.css" rel="stylesheet">

Usage

Start using EnexUI by adding classes to your HTML elements. Example:

<button class="btn btn-dark">Click me</button>

Components

EnexUI comes with the following built-in components:

  • Buttons
  • Cards
  • Grids
  • Navbar
  • Alerts

Sample Code

A simple EnexUI card component:


<div class="card">
  <div class="card-body">
    <h5 class="card-title">EnexUI Card</h5>
    <p class="card-text">Lightweight and responsive UI block.</p>
  </div>
</div>
